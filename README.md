  <h1>Angular Product Catalog (Material UI)</h1>

  <h2>Description</h2>
  <p>
    A single-page Angular application that displays product categories and products loaded from a local JSON file.
    The app uses Angular Material components (tabs, dialog, table) and lets users filter products by category
    through interactive UI controls.
  </p>

  <h2>Features</h2>
  <ul>
    <li>Imports product data from a JSON file and maps it into TypeScript interfaces</li>
    <li>Displays categories in a table using <code>*ngFor</code></li>
    <li>Shows products by category with button click events and filtering logic</li>
    <li>Uses Angular Material <strong>Tabs</strong> to separate UI sections</li>
    <li>Uses Angular Material <strong>Dialog</strong> to show a popup with product details</li>
    <li>Mobile-friendly layout (works well on smaller screens)</li>
  </ul>

  <h2>Tech Stack</h2>
  <ul>
    <li><strong>Angular</strong> &ndash; Components, data binding, events</li>
    <li><strong>TypeScript</strong> &ndash; Interfaces and application logic</li>
    <li><strong>Angular Material</strong> &ndash; Tabs, dialog, and UI components</li>
    <li><strong>HTML5</strong> &ndash; Angular templates</li>
    <li><strong>CSS3</strong> &ndash; Styling and layout</li>
    <li><strong>JSON</strong> &ndash; Local data source</li>
  </ul>

  <h2>What I Learned</h2>
  <ul>
    <li>How to structure data using TypeScript interfaces in Angular</li>
    <li>How to display lists and tables using <code>*ngFor</code> and event-driven UI</li>
    <li>Filtering data by category and updating UI state cleanly</li>
    <li>Using Angular Material components like tabs and dialogs</li>
    <li>Building a small but complete Angular feature flow (categories → products → details)</li>
  </ul>
